231. Power of Two

easy 2019-07-04

只需要判断二进制形式是否满足10..0的模式，0要单独处理。

if (!n) return false;
while (n) {
    if ((n & 1) && (n ^ 1)) return false; // 如果n的末尾为1且n!=1,那么说明不满足要求
    n >>= 1;
}
return true;
