66. Plus One

easy 2019-04-01

从后向前遍历数组，逐次进位，如果不需进位可以直接break退出。

最后判断是否需要在守卫增添一个新元素，如果需要，可以直接使用insert函数。

int carry = 1;
for (auto it = digits.rbegin(); it != digits.rend(); it++) {
    (*it) += carry; carry = (*it) / 10; (*it) %= 10;
    if (!carry) break;
}
if (carry) digits.insert(digits.begin(), 1); // 在 digits.begin() 的位置之前插入元素1
return digits;

67. Add Binary

``easy2019-04-01`

与上一题类似，上一题是加"1"，这里是加一个自定义的数。

仍然是从低位向高位累加，并且记得进位，在加完之后如果不再产生进位就退出。如果最终仍产生进位，那么在结果前添加字符"1"。

int carry = 0;
if (a.size() < b.size()) a.swap(b);
for (int i = b.size() - 1, j = a.size() - 1; j >= 0; i--, j--) {
    a[j] += ((i >= 0 ? b[i] : '0') - '0') + carry;
    carry = (a[j] - '0') / 2;
    a[j]  = (a[j] - '0') % 2 + '0';
    if (i < 0 && !carry) break;
}
if (carry) a.insert(a.begin(), '1');
return a;

69. Sqrt(x)

easy 2019-04-01

使用二分搜索的方法，寻找使得 a2≤xa^2\le xa2≤x 的最大的aaa

注意在取中间值时，使用 m=(h+l)/2m=(h+l)/2m=(h+l)/2 可能会溢出，因此应该使用 m=l+(h−l)/2m=l+(h-l)/2m=l+(h−l)/2

另外在判断aaa与xxx的关系时，直接进行a2a^2a2也可能溢出，因此应该比较aaa与x/ax/ax/a的关系

if (x == 0 || x == 1) return x;
int m, l = 1, h = x;
while (true) {
    m = ((h - l) >> 1) + l;
    if (m > x / m) h = m - 1;
    else if (m + 1 > x / (m + 1)) return m;
    else l = m + 1;
}
return x;

70. Climbing Stairs

easy 2019-04-01

使用DP算法。假设阶梯数为n，由于一次只能爬1或2，相当于在爬n-1层后又爬了1层，加上在爬了n-2层后又爬了2层，所以dp[n]=dp[n−1]+dp[n−2]dp[n]=dp[n-1]+dp[n-2]dp[n]=dp[n−1]+dp[n−2]，结果是Fibonacci数列。

if (n < 3) return n;
vector<int> dp(n + 1);
dp[1] = 1; dp[2] = 2;
for (int i = 3; i <= n; i++) dp[i] = dp[i - 1] + dp[i - 2];
return dp[n];
