1. Two Sum

easy 2019-02-01

  1. 排序 vector<int> a 时同时排序索引

    vector<int> idx(a.size());
iota(idx.begin(), idx.end(), 0);                                          // 生成从0开始大小为a.size的递增数组
sort(idx.begin(), idx.end(), [&a](int x, int y) { return a[x] < a[y]; }); // 根据数组a的大小排序索引数组
sort(a.begin(), a.end());

  2. 在有序数组 vector<int> a 中寻找等于数字 t 的元素的位置

    auto pos = equal_range(a.begin(), a.end(), t); // pos为pair类型，pos.first指向第一个不小于t的元素，pos.second指向第一个大于t的元素
int dis  = pos.second - pos.first;             // dis==0 找不到与t相等的元素；dis==1 恰有1个与t相等的元素；dis>1 有超过1个与t相等的元素                                              // 排序原数组a

7. Reverse Integer

easy 2019-02-01

int tmp, y = 0;
while (x) {
    tmp = y * 10 + x % 10;
    if ((x > 0 && (tmp / 10 < y)) || (x < 0 && (tmp / 10 > y)))
        return 0; // 根据x的正负判断是否溢出
    y = tmp;
    x /= 10;
}
return y;

这种计算方法类似于“秦九韶算法”，即anxn+an−1xn−1+⋯+a1x+a0=(((anx+an−1)x+an−2)x+⋯+a1)x+a0a_nx^n+a_{n-1}x^{n-1}+\cdots+a_1x+a_0=(((a_nx+a_{n-1})x+a_{n-2})x+\cdots+a_1)x+a_0an​xn+an−1​xn−1+⋯+a1​x+a0​=(((an​x+an−1​)x+an−2​)x+⋯+a1​)x+a0​

判断溢出：正整数在溢出时，由于INT_MAX+1=INT_MIN，因此溢出后一定会变小；类似的，负整数会变大

9. Palindrome Number

easy 2019-02-01

一个直接的思路是按照第7题将数字反转过来然后判断是否相等。另外：

  1. 将数字的最高位和最低位进行比较，然后是次高位和次低位，以此类推

    if (x < 0) return false;
int base = 1, tmp = x / 10;
while (tmp) { base *= 10; tmp /= 10; }
while (base > 1) {
    int last = x % 10, first = x / base; // 提取最低位和最高位的数字
    if (last != first) return false;
    x = (x / 10) % (base / 10); // 将x除去最高位和最低位
    base /= 100;
}
return true;

  2. 将数字的后半部分反转，然后与前半部分比较

    if (x < 0 || (x != 0 && x % 10 == 0)) return false;
int y = 0;
while (x > y) {
    y = 10 * y + x % 10; // 将x的后一半进行反转，存储在y中
    x /= 10;             // 将x的后一半去除
}
return (x == y || x == y / 10);
