53. Maximum Subarray

easy 2019-03-30

DP问题。从左至右，顺次计算数组元素的和，如果当前和小于0，那么就将和置0，即从当前元素重新计算；否则就持续累加，并保存最大值。

int sum = nums[0], max = sum;
for (size_t i = 1; i < nums.size(); i++) {
    sum = nums[i] + (sum < 0 ? 0 : sum);
    max = (max < sum ? sum : max);
}
return max;
