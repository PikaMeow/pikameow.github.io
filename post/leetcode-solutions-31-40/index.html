35. Search Insert Position

easy 2019-03-20

基本的二分查找。查找的是数组中“小于等于”target的最大的元素的下标。

int mid, beg = 0, end = nums.size(); // end指向数组后的位置
while (beg < end) {
    mid = (beg + end) >> 1;
    if (nums[mid] == target) return mid; // 一旦匹配就返回，可以提高一点效率，去掉也是可以的
    else if (nums[mid] < target) beg = mid + 1;
    else end = mid;
}
return beg;

当然也可以直接用lower_bound函数直接获取结果：

auto low = lower_bound(nums.begin(), nums.end(), target); // 返回指向“不小于”target的元素的迭代器
return low - nums.begin();

38. Count and Say

easy 2019-03-20

对于一个数，从高位向低位，连续有m个a，那么下一个数有2位就是ma，以此类推。

由于第i个数地生成仅依赖于第i-1个数，因此可以用递归的方式，先生成第i-1个数，然后再处理生成第i 个数。

if (n == 1) return "1";
string s = countAndSay(n - 1), res = "";
for (int i = 0; i < s.size(); i++) {
    int count = 1;
    while (i < s.size() - 1 && s[i] == s[i + 1])
        i++, count++;                    // 计算共有 count 个 s[i] 相邻
    res.append(to_string(count) + s[i]); // 将 count 和 s[i] 添加到下一个数中
}
return res;
