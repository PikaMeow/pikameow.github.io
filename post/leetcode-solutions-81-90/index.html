83. Remove Duplicates from Sorted List

easy 2019-04-10

建立一个工作指针p，然后遍历链表，比较相邻元素，如果相同就修改next指针。

ListNode *p = head;
while (p) { // 确保链表为空时不会出错
    if (p->next && p->val == p->next->val) p->next = p->next->next;
    else p = p->next; // 如果没有else，去重不完全
}
return head;

88. Merge Sorted Array

easy 2019-04-10

考虑到原地合并，从前向后会导致覆盖问题，因此从后向前合并即可。但要注意一些细节问题。

int i, j; // 由于循环条件与0有关，因此不能用size_t类型
for (i = m - 1, j = n - 1; i >= 0 && j >= 0;) {
    nums1[i + j + 2] = (nums1[i] > nums2[j] ? nums1[i--] : nums2[j--]);
    // 注意等式右边先执行，因此等式左边的下标应该是 根据右边执行后的下标 来计算
}
for (; j >= 0; j--) nums1[j] = nums2[j];
