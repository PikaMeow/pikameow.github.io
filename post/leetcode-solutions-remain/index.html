1046. Last Stone Weight

easy 2019-07-12

给定一个数组，每次从中取出2个最大的，然后将其相减后的绝对值再加回数组中，直到数组变空或者只剩下一个元素。可以通过最大堆来做，使用STL中的priority_queue即可。

另外，针对“取数组中最大的k个数”，可以使用 partial sort，即

partial_sort(stones.begin(), stones.begin() + k, stones.end(), greater<int>());
// 同样，取最小k个数只需要修改比较函数为 less<int>()

得到的前k个数就是按照降序排好的k个最大的数。

1078. Occurrences After Bigram

easy 2019-07-12

有2种思路，一种是将词作为基本单位，一种是将字符作为基本单位。对于前者，可以使用string类的操作可以很方便地完成查找。

vector<string> v,a;
stringstream s(text);  
string word; 
while (s >> word) a.push_back(word); // 构造为"词"的数组
for(int i=0;i<a.size()-1;i++)
  	if(a[i]==first && a[i+1]==second && i+2<a.size()) v.push_back(a[i+2]);
return v;

对于后者，使用基于字符的查找，就可以将first和second合并成一个字符串，并且使用已有的字符串匹配算法进行查找，这里使用了kmp算法。其中pattern = first + ' ' + second + ' '

void kmp_init(string pattern, vector<int> &next) {
  	next[0] = 0;
  	int k = 0, q = 1;
  	while (q < pattern.size()) {
    		while (k && pattern[k] != pattern[q]) k = next[k];
    if (pattern[k] == pattern[q]) k++;
    next[q++] = k;
  	}
}
void kmp(string text, string pattern, vector<string> &ans) {
    vector<int> next(pattern.size());
    kmp_init(pattern, next);
    int i = 0, k = 0;
    while (i < text.size()) {
        while (k && text[i] != pattern[k]) k = next[k - 1]; // 注意这里的 k-1
        if (text[i++] == pattern[k]) k++;
        if (k == pattern.size() && i < text.size()) {
            string tmp;
            for (int j = i; j < text.size() && text[j] != ' '; j++) tmp.push_back(text[j]);
            ans.push_back(tmp);
            k = next[k - 1];
      }
    }
}

1089. Duplicate Zeros

easy 2019-07-11

首先要确定有多少位被重复的0挤出了数组。我们可以使用2个指针，指针i按照下标顺次增加，指针j根据指针i的值，如果碰到0，就多加1，直到加到数组末尾为止。

之后开始从尾至头填充，需要分2种情况，即最后一个0重复之后是否还处在范围内。如果是，那么填充时仍然使用2个指针，指针i顺次减小，指针j根据指针i的值，如果碰到0，就多减1，并填充0，直到i==j，说明之前再没有0；如果不是，那么对最后一个0特殊处理，即在数尾端填充一个0，然后就不再考虑这个0，其余的按照上一种情况处理。

int i, j, len = arr.size();
for (i = 0, j = 0; i < len && j < len; i++, j++) if (arr[i] == 0) j++;
if (j > len) { j--; arr[--j] = arr[--i]; }
do {
  arr[--j] = arr[--i];
  if (arr[i] == 0) arr[--j] = 0;
} while (i != j);
