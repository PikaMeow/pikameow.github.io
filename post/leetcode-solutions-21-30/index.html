21. Merge Two Sorted Lists

easy 2019-03-10

直接在原来2个链表上进行操作，根据merge的顺序更改原链表的指针。注意修改指针的顺序。可以通过new一个dummy节点来作为新链表的头。

ListNode *dummy = new ListNode(0), *p = dummy; // 定义头指针和工作指针
while (l1 && l2) {
    if (l1->val <= l2->val) { p->next = l1; l1 = l1->next; } 
    else { p->next = l2; l2 = l2->next;}
    p = p->next;
}
if (l1) p->next = l1;
if (l2) p->next = l2;
return dummy->next;

22. Generate Parentheses

medium 2019-07-20

与17题类似，字符串回溯，使用DFS即可。注意已生成的字符串中，任意前k个字符，左括号的数目都不小于右括号。

vector<string> generateParenthesis(int n) {
    vector<string> ans;
    dfs(n, n, "", ans);
    return ans;
}
void dfs(int left, int right, string cur, vector<string> &ans) {
    if (right == 0) { ans.push_back(cur); return; }
    if (left < right) dfs(left, right - 1, cur + ')', ans); // 剩余右括号一定不能小于左括号
    if (left > 0) dfs(left - 1, right, cur + '(', ans); // 左括号消耗完之前都可以继续加
}

26. Remove Duplicates From Sorted Array

easy 2019-03-10

基本思想是处理后的数组各元素的下标一定小于等于原数组各元素的下标。因此可以从左至右逐个比对，然后将新元素直接覆盖原有位置的元素。

除了手动实现之外，还可以使用unique函数，将容器的[first,last)之间的元素唯一化，并且返回指向新的末尾元素的迭代器，复杂度与原数组的大小呈线性。

auto i = unique(nums.begin(), nums.end()); // 处理后的末尾位置
auto j = nums.begin();                     // 处理前(后)的起始位置
return i - j;

28. Implement strStr()

easy 2019-03-10

判断字符串needle是否为字符串haystack的子串。

如果是，返回第一次出现的索引；如果不是，返回-1；如果needle为空，返回0。

方法一：KMP算法 O(n+m)O(n+m)O(n+m)
void kmp_init(string &needle, vector<int> &next) { // 初始化 next 数组
    next[0] = 0;
    int k   = 0;
    for (int q = 1; q < needle.length(); q++) {
        while (k && needle[k] != needle[q]) k = next[k - 1];
        if (needle[k] == needle[q]) k++;
        next[q] = k;
    }
    return next;
}
int kmp(string &haystack, string &needle) { // 进行字符串匹配
    if (needle.empty()) return 0;
    vector<int> next(needle.length(), 0);
    kmp_init(needle, next);
    int k = 0;
    for (int q = 0; q < haystack.length(); q++) {
        while (k && needle[k] != haystack[q]) k = next[k - 1];
        if (needle[k] == haystack[q]) k++;
        if (k == needle.length()) return q - k + 1; // 返回第1次出现位置的索引
    }
    return -1;
}
方法二：字符串Hash O(n+m)O(n+m)O(n+m)

将字符串needle中的字符映射到map中，然后遍历字符串haystack找到与needle中各字符都能匹配得上的相同数量的字符，作为候选串，然后再逐个字符比较，如果顺序也对得上，那么就返回相应索引。

（就本题而言，该方法与 Brute-Force 方法效率差别不大）

if (needle.empty()) return 0;
unordered_map<char, int> m;
for (auto c : needle) m[c]++;                // 字符串映射
int count = needle.size(), beg = 0, end = 0; // 双指针，保存候选串的起始和末尾位置
while (end < haystack.length()) {
    if (m[haystack[end++]]-- > 0) count--; // count用来判断当前已经匹配上了多少字符
    if (end - beg == needle.length()) {
        if (count == 0) { // 如果恰好找到了一个候选，那么逐字符比较
            int i = beg, j = 0;
            while (j < needle.length() && haystack[i] == needle[j]) i++, j++;
            if (j == needle.length()) return beg;
        }
        if (m[haystack[beg++]]++ >= 0)
            count++; // 如果发现顺序不对，那么后移一位继续判断，恢复相应字符在映射m中的值
    }
}
return -1;

29. Divide Two Integers

medium 2019-07-20

使用位操作实现两数相除。我们采用这样的思想：链接 一个数字在二进制下总可以表达为q=ai2i+⋯+a020q=a_i2^i+\cdots+a_02^0q=ai​2i+⋯+a0​20的形式，例如15/3=5=(101)2=22+2015/3=5=(101)_2=2^2+2^015/3=5=(101)2​=22+20，即15=22×3+20×315=2^2\times 3+2^0\times 315=22×3+20×3，因此我们可以采取如下的思路，将divisor不断左移一位，直到再移就超过dividend为止，然后用dividend减去，例如3 << 2 = 12, 15 - 12 = 3，用新的数作为新的dividend，重复上面的过程，直到dividend = 0，例如3 << 0 = 3, 3 - 3 = 0。此时每次移的数位相加：1 << 2 + 1 << 0 = 5即为所求的商。

有了这样的思想之后，我们可以实现代码如下：

当divisor=0时将出现溢出；当divisor=-1且dividend=INT_MIN时也会溢出；当divisor=1且dividend==INT_MIN时，由于计算过程中会现将dividend变为正，结果也会出现溢出。因此先处理这3种情况。

if (divisor == 0) return INT_MAX;
if (divisor == 1) return dividend;
if (divisor == -1) return dividend == INT_MIN ? INT_MAX : -dividend;

因为我们在将divisor左移的过程中，要使其大于dividend，因此很可能出现溢出的情况，所以我们使用long long类型来保存计算的中间变量防止溢出。代码如下：

long long nom = dividend, den = divisor;
nom = nom < 0 ? -nom : nom;
den = den < 0 ? -den : den;
int quo = 0;
while (nom >= den) {
    long long tmp = den, m = 1;
    while ((tmp << 1) <= nom) { tmp <<= 1; m <<= 1; } // 尝试左移，直到大于被除数为止
    nom -= tmp; quo |= m;
}
return (dividend < 0 ^ divisor < 0) ? -quo : quo; // 异号时取负

这种利用更大的存储空间的行为有作弊之嫌，假如要求两个long long类型数字的除法就没有办法了，因此我们想只使用int类型的存储空间来完成这个运算。

为了避免出现中间结果溢出的情况，我们首先对divisor=INT_MIN的情况做处理，它会在变为正数时中溢出。然后对于dividend=INT_MIN时可能出现的溢出，我们将其加上abs(divisor)，并对商增1。之后在divisor左移的过程中，我们通过增加(divisor<<1) > 0这个条件来判断是否溢出。

if (divisor == INT_MIN) return dividend == INT_MIN ? 1 : 0;
int sign = dividend < 0 ^ divisor < 0, quo = 0;
divisor = divisor < 0 ? -divisor : divisor;
if (dividend == INT_MIN) { dividend += divisor; quo++;}
dividend = dividend < 0 ? -dividend : dividend;
while (dividend >= divisor) {
    int d = dividend, r = divisor, m = 1;
    while ((r << 1) > 0 && (r << 1) <= d) // 如果r即将溢出，那么 r<<1 的符号位会变为1，被解释为负数
    		{ r <<= 1; m <<= 1; } 
    dividend -= r; quo += m;
}
return sign ? -quo : quo;
