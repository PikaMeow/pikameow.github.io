111. Minimum Depth of Binary Tree

easy 2019-07-01

与计算最大深度很类似，只需将max变为min。但需注意一点，如果左右子树一个为空，一个非空，那么最小高度是不计算空子树的，只计算非空的一侧子树。

if (!root) return 0;
int lh = minDepth(root->left), rh = minDepth(root->right);
// 方法一：分别判断一侧子树为空的情况
if ((!lh && rh)) return rh + 1;
else if ((lh && !rh)) return lh + 1;
else return min(lh, rh) + 1;
// 方法二：更简洁！
return 1 + (min(lh, rh) ? min(lh, rh) : max(lh, rh));

112. Path Sum

easy 2019-07-01

DFS即可。相比于每次递归都记录一个值，然后到叶子节点时与target sum比较，示例解法使用了递归减的方式，最后结果直接与0相比，避免了使用一个参数保存中间结果，更加简洁。

if(root == NULL)  return false;
int newsum = sum - root->val;
if(root->left == NULL && root->right == NULL)  return newsum == 0;
return hasPathSum(root->left, newsum) || hasPathSum(root->right, newsum);

118. Pascal's Triangle

easy 2019-07-01

首先注意数字的存放次序。可以选择直接建立二维数组，然后使用DP的方法向里面填充数据；同时，考虑到每一行都是由上一行直接决定的，因此也可以使用一个一维数组来做运算，之后复制到二维数组中。

for (int k = 0; k < numRows; k++) {
    row.push_back(1);
    for (int i = k - 1; i > 0; i--) {
        row[i] = row[i] + row[i - 1];
    }
    tri.push_back(row);
}
