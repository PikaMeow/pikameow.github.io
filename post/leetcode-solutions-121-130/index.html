125. Valid Palindrome

easy 2019-07-02

熟悉字符串操作的题目。使用迭代器时，a=s.begin()和b=s.rbegin()由于类型不同，是不能进行比较的。

头文件<cctype>中有三个函数isalpha(), isdigit(), isalnum()分别用来判断输入字符是否为字母、数字、字母及数字。

for (auto it1 = s.begin(), it2 = s.end() - 1; it1 < it2; it1++, it2--) {
    while (!isalnum(*it1) && it1 < it2)  it1++;
    while (!isalnum(*it2) && it1 < it2)  it2--;
    if (tolower(*it1) != tolower(*it2))  return false;
}
return true;
