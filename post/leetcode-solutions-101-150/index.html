101. Symmetric Tree

easy 2019-04-10

与上一题类似，相当于判断两个子树是否镜像相同，注意区分所需要比较的子树的节点。

bool isSubSymmetric(TreeNode *left, TreeNode *right) {
    if ((left && !right) || (!left && right)) return false;
    if (!(left && right)) return true;
    if (left->val == right->val)
        return isSubSymmetric(left->right, right->left) && isSubSymmetric(left->left, right->right); // 注意要进行比较的子树节点
    return false;
}

104. Maximum Depth of Binary Tree

easy 2019-07-01

遍历所有的分支，找到深度的最大值并返回。可以递归来获取子树的最大深度。

int maxDepth(TreeNode *root) {
    return root ? 1 + max(maxDepth(root->left), maxDepth(root->right)) : 0;
}

107. Binary Tree Level Order Traversal II

easy 2019-07-01

如何确定数组的大小？一可以使用上题的方法先算出树的最大深度，然后向相应深度的节点数组中填充数据；二可以在递归向下的过程中，如果发现深度超过数组范围，则动态增大数组长度。

在创建空的vector<int>数组时，不需要声明，直接使用a.push_back({})即可。

如果使用方法二，那么在算法结束后需要将数组进行倒序，可以使用reverse(a.begin(), a.end())的方法。方法一在效率上更快一些。核心代码如下：

void traversal(TreeNode *p, int depth, vector<vector<int>> &tree) {
    if (!p) return;
    if (depth >= tree.size()) tree.push_back({});
    tree[depth].push_back(p->val);
    traversal(p->left, depth + 1, tree);
    traversal(p->right, depth + 1, tree);
}

108. Convert Sorted Array to Binary Search Tree

easy 2019-07-01

本来以为要实现BST的平衡操作，后来发现只需要按照BST的构造方式将元素插入即可。另外由于CPP中的重载，递归版本的函数可以直接采用相同的函数名。

TreeNode *sortedArrayToBST(vector<int> &nums, int beg, int end) {
    if (beg >= end) return NULL;
    int mid     = (beg + end) >> 1;
    TreeNode *p = new TreeNode(nums[mid]);
    p->left     = sortedArrayToBST(nums, beg, mid);
    p->right    = sortedArrayToBST(nums, mid + 1, end);
    return p;
}

110. Balanced Binary Tree

easy 2019-07-01

思路是计算每个节点的左右高度差，然后返回是否平衡。需要注意的是：只要有任一节点高度差大于1，即不平衡，而非仅考虑根节点。因此如果检测到错误，最好能够在递归中途就直接返回。

自己代码使用了-1作为不合法的高度值来表示检测到存在不平衡的情况(不是很美)。示例解法中，使用一个引用参数来保存高度值，看起来更整洁，如下所示：

bool isBalanced(TreeNode *root) {
    int height = 0;
    return checkBalance(root, height);
}
bool checkBalance(TreeNode *root, int &height) {
    if (root == NULL) { height = 0; return true; }
    int leftHeight  = 0;
    int rightHeight = 0;
    if (!checkBalance(root->left, leftHeight)) return false;
    if (!checkBalance(root->right, rightHeight)) return false;
    if (abs(leftHeight - rightHeight) > 1) return false;
    height = max(leftHeight, rightHeight) + 1;
    return true;
}

111. Minimum Depth of Binary Tree

easy 2019-07-01

与计算最大深度很类似，只需将max变为min。但需注意一点，如果左右子树一个为空，一个非空，那么最小高度是不计算空子树的，只计算非空的一侧子树。

if (!root) return 0;
int lh = minDepth(root->left), rh = minDepth(root->right);
// 方法一：分别判断一侧子树为空的情况
if ((!lh && rh)) return rh + 1;
else if ((lh && !rh)) return lh + 1;
else return min(lh, rh) + 1;
// 方法二：更简洁！
return 1 + (min(lh, rh) ? min(lh, rh) : max(lh, rh));

122. Best Time to Buy and Sell Stock II

贪心问题。由于可以多次买入和卖出，因此对于任意相邻2次，在预知到有收益的时候都提前买入，预知到亏损的时候提前卖出，最后得到的一定是最大的收益。

int ans = 0;
for (int i = 0; i < prices.size() - 1; i++) {
    int diff = prices[i + 1] - prices[i];

125. Valid Palindrome

easy 2019-07-02

熟悉字符串操作的题目。使用迭代器时，a=s.begin()和b=s.rbegin()由于类型不同，是不能进行比较的。

头文件<cctype>中有三个函数isalpha(), isdigit(), isalnum()分别用来判断输入字符是否为字母、数字、字母及数字。

for (auto it1 = s.begin(), it2 = s.end() - 1; it1 < it2; it1++, it2--) {
    while (!isalnum(*it1) && it1 < it2)  it1++;
    while (!isalnum(*it2) && it1 < it2)  it2--;
    if (tolower(*it1) != tolower(*it2))  return false;
}
return true;

141. Linked List Cycle

easy 2019-07-02

方法一：两个指针追赶法

使用2个指针p和q指向链表头，然后p每次前进2步，q每次前进1步，如果p到达链表尾，那么说明没有环；每次前进后判断(p==q || p==q.next)，如果为真，表示p通过环追赶上了q，说明有环。

时间效率分析：假设长度为nnn的链表有环，其中无环部分长度为aaa，有环部分为bbb。对指针qqq来说，到进入环的时刻，耗时为O(a)O(a)O(a)，此时ppp已经在环中。然后变成指针的追赶问题，两者的最大距离为bbb，而速度差为111，因此ppp追上qqq耗时最多为O(b)O(b)O(b)，加起来为O(n)O(n)O(n)。

ListNode *p = head->next, *q = head;
while (p) {
    p = p->next;  q = q->next;
    if (!p || !(p->next))  break;
    p = p->next;
    if (p == q || p == q->next)  return true; // 仅用 p==q 也可，但可能会更慢
}
return false;

方法二：倒转链表法

首先将链表倒转过来，如果链表中有环，那么倒转之后的rhead指针与原来的head指针相同。需要注意的是，判断是要保证链表元素大于1(含循环)。

ListNode *reverse(ListNode *head) {
    ListNode *p = NULL, *q = NULL; // prev and follow pointer
    while (head) {
        q          = head->next; // 找到下一个元素
        head->next = p; // 将当前head的next反向
        p          = head; // 将当前head置为prev
        head       = q; // 移动至下一个元素
    }
    return p;
}

方法三：哈希映射法

使用数据结构unordered_map，即用哈希表来保存链表的每个指针。然后遍历列表，如果发现键已经存在表中，说明链表有环。

unordered_map<ListNode *, int> table; // 键有用，值无用，可以随意设置
while (head) {
    table.insert(make_pair(head, head->val));
    head = head->next;
    if (table.find(head) != table.end())  return true;
}
return false;
