53. Maximum Subarray

easy 2019-03-30

DP问题。从左至右，顺次计算数组元素的和，如果当前和小于0，那么就将和置0，即从当前元素重新计算；否则就持续累加，并保存最大值。

int sum = nums[0], max = sum;
for (size_t i = 1; i < nums.size(); i++) {
    sum = nums[i] + (sum < 0 ? 0 : sum);
    max = (max < sum ? sum : max);
}
return max;

66. Plus One

easy 2019-04-01

从后向前遍历数组，逐次进位，如果不需进位可以直接break退出。

最后判断是否需要在守卫增添一个新元素，如果需要，可以直接使用insert函数。

int carry = 1;
for (auto it = digits.rbegin(); it != digits.rend(); it++) {
    (*it) += carry; carry = (*it) / 10; (*it) %= 10;
    if (!carry) break;
}
if (carry) digits.insert(digits.begin(), 1); // 在 digits.begin() 的位置之前插入元素1
return digits;

67. Add Binary

``easy2019-04-01`

与上一题类似，上一题是加"1"，这里是加一个自定义的数。

仍然是从低位向高位累加，并且记得进位，在加完之后如果不再产生进位就退出。如果最终仍产生进位，那么在结果前添加字符"1"。

int carry = 0;
if (a.size() < b.size()) a.swap(b);
for (int i = b.size() - 1, j = a.size() - 1; j >= 0; i--, j--) {
    a[j] += ((i >= 0 ? b[i] : '0') - '0') + carry;
    carry = (a[j] - '0') / 2;
    a[j]  = (a[j] - '0') % 2 + '0';
    if (i < 0 && !carry) break;
}
if (carry) a.insert(a.begin(), '1');
return a;

70. Climbing Stairs

easy 2019-04-01

使用DP算法。假设阶梯数为n，由于一次只能爬1或2，相当于在爬n-1层后又爬了1层，加上在爬了n-2层后又爬了2层，所以dp[n]=dp[n−1]+dp[n−2]dp[n]=dp[n-1]+dp[n-2]dp[n]=dp[n−1]+dp[n−2]，结果是Fibonacci数列。

if (n < 3) return n;
vector<int> dp(n + 1);
dp[1] = 1; dp[2] = 2;
for (int i = 3; i <= n; i++) dp[i] = dp[i - 1] + dp[i - 2];
return dp[n];

83. Remove Duplicates from Sorted List

easy 2019-04-10

建立一个工作指针p，然后遍历链表，比较相邻元素，如果相同就修改next指针。

ListNode *p = head;
while (p) { // 确保链表为空时不会出错
    if (p->next && p->val == p->next->val) p->next = p->next->next;
    else p = p->next; // 如果没有else，去重不完全
}
return head;
