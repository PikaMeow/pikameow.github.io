101. 对称二叉树

easy 2019-04-10

与上一题类似，相当于判断两个子树是否镜像相同，注意区分所需要比较的子树的节点。

bool isSubSymmetric(TreeNode *left, TreeNode *right) {
    if ((left && !right) || (!left && right)) return false;
    if (!(left && right)) return true;
    if (left->val == right->val)
        return isSubSymmetric(left->right, right->left) && isSubSymmetric(left->left, right->right); // 注意要进行比较的子树节点
    return false;
}

104. 二叉树的最大深度

easy 2019-07-01

遍历所有的分支，找到深度的最大值并返回。可以递归来获取子树的最大深度。

int maxDepth(TreeNode *root) {
    return root ? 1 + max(maxDepth(root->left), maxDepth(root->right)) : 0;
}

107. 二叉树的层次遍历 II

easy 2019-07-01

如何确定数组的大小？一可以使用上题的方法先算出树的最大深度，然后向相应深度的节点数组中填充数据；二可以在递归向下的过程中，如果发现深度超过数组范围，则动态增大数组长度。

在创建空的vector<int>数组时，不需要声明，直接使用a.push_back({})即可。

如果使用方法二，那么在算法结束后需要将数组进行倒序，可以使用reverse(a.begin(), a.end())的方法。方法一在效率上更快一些。核心代码如下：

void traversal(TreeNode *p, int depth, vector<vector<int>> &tree) {
    if (!p) return;
    if (depth >= tree.size()) tree.push_back({});
    tree[depth].push_back(p->val);
    traversal(p->left, depth + 1, tree);
    traversal(p->right, depth + 1, tree);
}

108. 将有序数组转换为二叉搜索树

easy 2019-07-01

本来以为要实现BST的平衡操作，后来发现只需要按照BST的构造方式将元素插入即可。另外由于CPP中的重载，递归版本的函数可以直接采用相同的函数名。

TreeNode *sortedArrayToBST(vector<int> &nums, int beg, int end) {
    if (beg >= end) return NULL;
    int mid     = (beg + end) >> 1;
    TreeNode *p = new TreeNode(nums[mid]);
    p->left     = sortedArrayToBST(nums, beg, mid);
    p->right    = sortedArrayToBST(nums, mid + 1, end);
    return p;
}

110. 平衡二叉树

easy 2019-07-01

思路是计算每个节点的左右高度差，然后返回是否平衡。需要注意的是：只要有任一节点高度差大于1，即不平衡，而非仅考虑根节点。因此如果检测到错误，最好能够在递归中途就直接返回。

自己代码使用了-1作为不合法的高度值来表示检测到存在不平衡的情况(不是很美)。示例解法中，使用一个引用参数来保存高度值，看起来更整洁，如下所示：

bool isBalanced(TreeNode *root) {
    int height = 0;
    return checkBalance(root, height);
}
bool checkBalance(TreeNode *root, int &height) {
    if (root == NULL) { height = 0; return true; }
    int leftHeight  = 0;
    int rightHeight = 0;
    if (!checkBalance(root->left, leftHeight)) return false;
    if (!checkBalance(root->right, rightHeight)) return false;
    if (abs(leftHeight - rightHeight) > 1) return false;
    height = max(leftHeight, rightHeight) + 1;
    return true;
}

111. 二叉树的最小深度

easy 2019-07-01

与计算最大深度很类似，只需将max变为min。但需注意一点，如果左右子树一个为空，一个非空，那么最小高度是不计算空子树的，只计算非空的一侧子树。

if (!root) return 0;
int lh = minDepth(root->left), rh = minDepth(root->right);
// 方法一：分别判断一侧子树为空的情况
if ((!lh && rh)) return rh + 1;
else if ((lh && !rh)) return lh + 1;
else return min(lh, rh) + 1;
// 方法二：更简洁！
return 1 + (min(lh, rh) ? min(lh, rh) : max(lh, rh));

124. 二叉树中的最大路径和

hard 2019-03-18

思路一：递归

这个题目与计算一棵二叉树的最大高度类似。采用分治的思想，假设最长的路径以某个结点node为最低结点(距离根节点最近)，那么最长路径长度为其左右子树的最长路径之和加上node本身的值。另外还要注意，题目中并没有限定路径的起点和终点都是叶节点，因此在取左右子树的路径长度时，如果小于0，就直接用0替代，即抛弃该侧的路径。

int maxPathSum(TreeNode *node, int &ans) {
    if (node == NULL) return 0;
    int left  = max(0,maxPathSum(node->left, ans));
    int right = max(0,maxPathSum(node->right, ans));
    ans       = max(ans, left + right + node->val);
    return max(left, right) + node->val; // 返回的时候是挑选左右子树中路径的最大值加上当前节点值
}
int maxPathSum(TreeNode *root) {
    int ans = root == NULL ? 0 : root->val;
    maxPathSum(root, ans);
    return ans;
}

144. 二叉树的前序遍历

给定一个二叉树，返回它的 前序 遍历。请用迭代的方式实现。

思路一：栈和数组

将每个结点加入栈，然后依次弹出加入到数组头（头结点先输出），并且将该结点的右结点和左结点（更靠近栈顶 先遍历左子树）依次加入栈。

stack<TreeNode *> s;
vector<int> ans;
if (!root) return ans;
s.push(root);
while (!s.empty()) {
    TreeNode *p = s.top();
    s.pop();
    ans.push_back(p->val);
    if (p->right) s.push(p->right);
    if (p->left) s.push(p->left); // 左子树更靠近栈顶，首先被取出
}
return ans;

94. 二叉树的中序遍历

给定一个二叉树，返回它的中序 遍历。请用迭代的方式实现。

思路一：栈和数组

stack<TreeNode *> s;
vector<int> ans;
TreeNode *cur = root;
while (!s.empty() || cur != NULL) {
    while (cur) {
        s.push(cur);
        cur = cur->left;
    }
    TreeNode *p = s.top();
    s.pop();
    ans.push_back(p->val);
    if (p->right) {
        cur = p->right;
    }
}
return ans;

145. 二叉树的后序遍历

给定一个二叉树，返回它的 后序 遍历。请用迭代的方式实现。

思路一：栈和链表

将每个结点加入栈，然后依次弹出加入到链表头（头结点最后输出），并且将该结点的左结点和右结点（更靠近栈顶 在链表靠后位置）依次加入栈。最后将链表变成一个数组返回即可。

stack<TreeNode *> s;
list<int> l;
if (!root) return vector<int>();
s.push(root);
while (!s.empty()) {
    TreeNode *p = s.top();
    s.pop();
    l.push_front(p->val);
    if (p->left) s.push(p->left);
    if (p->right) s.push(p->right); // 后加入更靠近栈顶，弹出后在链表靠后的位置
}
return vector<int>(l.begin(), l.end());

572. 另一个树的子树

给定两个非空二叉树 s 和 t，检验 s 中是否包含和 t 具有相同结构和节点值的子树。s 的一个子树包括 s 的一个节点和这个节点的所有子孙。s 也可以看做它自身的一棵子树。

思路一：序列化 + KMP

首先将树通过序列化将其结构唯一地表示出来，可以通过先序或者后序序列化的方法。之后在通过字符串匹配算法看是否存在包含关系，如果存在，那么说明是子树。

string serialize(TreeNode *s) { // 先序序列化，迭代方式
    stack<TreeNode *> st;
    string a;
    st.push(s);
    while (!st.empty()) {
        TreeNode *p = st.top();
        st.pop();
        if (p) {
            a += to_string(p->val) + "_";
            st.push(p->right);
            st.push(p->left);
        } else { a += "#_"; }
    }
    return a;
}
vector<int> getNext(string t) { // 计算next数组
    vector<int> next(t.size());
    next[0] = -1;
    if (t.size() == 1) return next;
    next[1] = 0;
    int cn  = 0;
    for (int i = 2; i < t.size();) {
        if (t[i - 1] == t[cn]) { next[i++] = ++cn; }
        else if (next[cn] != -1) { cn = next[cn]; }
        else { next[i++] = 0; }
    }
    return next;
}
int kmp(string s, string t) {
    if (t.size() == 0) return 0;
    auto next = getNext(t);
    int i1 = 0, i2 = 0; // 抽象成两个指针的移动
    while (i1 < s.size() && i2 < t.size()) {
        if (s[i1] == t[i2]) { i1++; i2++; }
        else if (next[i2] != -1) { i2 = next[i2]; }
        else { i1++; }
    }
    return i2 == t.size() ? i1 - i2 : -1;
}
bool isSubtree(TreeNode *s, TreeNode *t) {
    string a = serialize(s);
    string b = serialize(t);
    return kmp(a, b) != -1;
}
