202. Happy Number

easy 2019-07-09

参考Wiki。对于m(m>3)位的数，在进行一次平方加和后，所得到的数最大为 81m；即所有超过1000的数，在若干次平方加和后都会小于1000。而小于1000的最大数为999，接下来发现所有3位数都会变成至多2位数。而对于任意小于100的数，经过逐个计算，要么它收敛到1，要么进入4→⋯→44\rightarrow \cdots \rightarrow 44→⋯→4的循环，因此可以通过这个条件进行判断。

while (n != 1 && n != 4) {
    int res = 0;
    while (n) { res += (n % 10) * (n % 10); n /= 10; }
    n = res;
}
return n == 1 ? true : false;

当然，如果不使用这个性质，还可以使用一个Hash表保存访问过的数，如果出现了循环，且并不为1，那么就返回false。

203. Remove Linked List Elements

easy 2019-07-03

对于该题而言，链表中的元素分三类：开头，中间，结尾。其中开头元素要单独处理，用以移动head的位置，另两类可以一并处理，但要注意边界情况。

while(head && head->val==val) head=head->next;
ListNode *p=head;
while(p && p->next) {
    if(p->next->val==val) p->next=p->next->next;
    else p=p->next;
}
return head;

206. Reverse Linked List

easy 2019-07-04

非递归的方式在检测链表是否有环的第#141题中已经实现，这里使用递归的方法，效果相同，写起来更简洁。

ListNode *reverseList(ListNode *p, ListNode *q) { // 初始调用时，p=head, q=NULL
    if (!p)  return q;
    ListNode *k = p->next;  p->next = q;
    return reverseList(k, p);
}
